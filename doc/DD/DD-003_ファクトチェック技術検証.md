# DD-003: ファクトチェック・技術検証

| 作成日 | 更新日 | ステータス |
|--------|--------|------------|
| 2026-03-20 | 2026-03-20 | 完了 |

## 目的

Geminiが生成したリサーチドキュメント内のAPI名・料金・モデル名・コードサンプルの正確性を検証し、公式ドキュメントのURLを引用として付与する。

## 背景・課題

- 全ドキュメントがGeminiとのチャットから生成されており、ファクトの正確性が未検証
- API名やモデル名が古い可能性がある（例: `text-multilingual-embedding-002` は最新か？）
- 料金の算出根拠が不明確（公式料金表へのリンクがない）
- コードサンプルが動作するか未確認

## 検討内容

### 検証対象カテゴリ

| カテゴリ | 検証内容 | 対象ファイル |
|---------|---------|-------------|
| **GCPサービス名** | Document AI, Firestore Vector Search, Vertex AI Ranking API 等の正式名称・提供状況 | 全ファイル |
| **モデル名** | `text-multilingual-embedding-002`, Gemini 1.5 Flash/Pro 等の最新モデル名 | 01, 03, 03-2 |
| **料金** | Document AI, Embedding API, Gemini, Ranking API の最新料金 | 01-2 |
| **コードサンプル** | Firestore Vector Search のクエリ構文、Vertex AI SDK の呼び出し方 | 02-2, 03 |
| **アルゴリズム** | RRF の数式、HNSW パラメータの説明 | 03-2 |
| **Genkit API** | `defineTool`, `Flow`, `run`, Evaluators の最新API | 02-2, 05, 07 |

### やらないこと
- コードサンプルの実行環境構築（別DDで対応）
- 技術内容の加筆（事実の正誤判定のみ）
