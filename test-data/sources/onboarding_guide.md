# 新入社員オンボーディングガイド

制定日: 2024-04-01
最終改訂: 2025-10-01
管理部門: 人事部（渡辺裕子 ※人事関連）/ IT部門（石井翔太 ※IT関連）
閲覧権限: 全社員
問い合わせ: 人事部 内線4100 / IT部門 内線1111（helpdesk@example.com）

---

## はじめに

本ガイドは、新入社員が入社初日からスムーズに業務を開始できるよう、必要な手続き・研修・環境設定の流れをまとめたものです。不明な点があれば、配属部門のメンターまたは各担当部門にお気軽にお問い合わせください。

---

## 入社初日のタスク

入社日の午前中に以下の手続きを完了してください。

### 総務部での手続き（内線4200）
- [ ] 社員証の受取（顔写真付きICカード）
- [ ] ロッカーキーの受取（東京本社の場合は3F更衣室）
- [ ] オフィスフロアマップ・避難経路の説明

### 人事部での手続き（内線4100）
- [ ] 雇用契約書への署名
- [ ] 社会保険・雇用保険の加入手続き
- [ ] マイナンバー届出書の提出
- [ ] 給与振込口座の届出
- [ ] 就業規則・各種規程の配布と説明

### IT部門での手続き（内線1111 / helpdesk@example.com）
- [ ] 業務用PCの受取・初期設定
- [ ] Microsoft 365アカウントの有効化（メール・Teams・OneDrive）
- [ ] GlobalProtect VPNの設定（詳細はVPN接続マニュアルを参照）
- [ ] 社内ポータルへのログイン確認
- [ ] 複合機の認証カード登録

### 午後
- [ ] 配属部門への挨拶
- [ ] メンターとの初回面談（自己紹介・今後のスケジュール確認）
- [ ] デスク・備品の確認

## 1週目のタスク

| タスク | 担当部門 | 期限 | 備考 |
|-------|---------|------|------|
| 情報セキュリティ研修（eラーニング） | IT部門 | 入社3日以内 | 社内ポータル「研修」から受講。修了テスト合格が必須 |
| コンプライアンス研修 | 法務部 | 入社5日以内 | 社内ポータル「研修」から受講 |
| 部門紹介・挨拶回り | 配属部門 | 入社1週間以内 | メンターが同行 |
| メンター面談（2回目） | 配属部門 | 入社5日目 | 業務内容の詳細説明・質疑応答 |
| 交通費定期券申請 | 総務部 | 入社1週間以内 | 社内ポータル「経費精算」から申請 |
| 名刺発注 | 総務部 | 入社3日以内 | 初回50枚。社内ポータル「総務申請」から発注 |

---

## 1ヶ月目のタスク

- [ ] **OJT開始**: 配属部門のOJT計画に基づき、実務を通じた研修を開始
- [ ] **有給休暇制度の説明**: 入社6ヶ月後に10日付与。詳細は年次有給休暇規程を参照
- [ ] **目標設定面談**: 上長と今期の目標を設定（社内ポータル「人事評価」から入力）
- [ ] **1ヶ月振り返り面談**: メンター・上長と入社後の振り返りを実施
- [ ] **福利厚生説明会**: 人事部主催（月1回開催、日程は社内ポータルで案内）

## 緊急連絡先

- **IT障害・セキュリティインシデント**: IT部門 内線1100 / security@example.com
- **体調不良・労災**: 人事部 内線4100
- **火災・地震等**: 総務部 内線4200（東京本社の防災担当）
- **ハラスメント相談窓口**: 人事部 内線4150（外部窓口の案内あり）
