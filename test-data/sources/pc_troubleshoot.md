# PCトラブルシューティングFAQ

よくある質問と対処法をまとめています。解決しない場合はITヘルプデスク（内線: 1111）にお問い合わせください。

---

## Q1: PCの動作が遅い（重い）

**原因**: メモリ不足またはディスク容量不足の可能性が高いです。

**対処法**:
1. タスクマネージャー（Ctrl+Shift+Esc）を開き、メモリ使用率を確認
2. メモリ使用率が80%以上の場合、不要なアプリケーションを終了
3. ディスクの空き容量が10GB以下の場合、不要なファイルを削除するか、ファイルサーバーに移動
4. 上記で改善しない場合、PCの再起動を実施（週に1回の再起動を推奨）

## Q2: Outlookが起動しない

**原因**: プロファイルの破損またはアップデートの不整合。

**対処法**:
1. Outlookをセーフモードで起動: Windowsキー+R → `outlook.exe /safe` を実行
2. セーフモードで起動できた場合、アドインを全て無効化して再起動
3. セーフモードでも起動できない場合、コントロールパネル → メール → プロファイルを削除して再設定

## Q3: プリンターで印刷できない

**対処法**:
1. プリンターの電源が入っているか、用紙切れやインク切れがないか確認
2. 印刷キューに溜まったジョブを全て削除（プリンターアイコン右クリック → 全ドキュメントの取り消し）
3. プリンタードライバーを再インストール: 社内ポータル → ダウンロード → プリンタードライバー

## Q4: Wi-Fiに接続できない

**対処法**:
1. 機内モードがオンになっていないか確認
2. Wi-Fiを一度オフにしてからオンに戻す
3. 社内Wi-FiのSSID「CORP-WIFI-5G」に接続（「CORP-WIFI-GUEST」は来客用のため社内リソースにアクセス不可）
4. それでも接続できない場合、ネットワークのリセット: 設定 → ネットワーク → ネットワークのリセット

## Q5: 画面がフリーズした

**対処法**:
1. まず2〜3分待つ（バックグラウンド処理中の可能性）
2. Ctrl+Alt+Delete でタスクマネージャーを開き、応答なしのプログラムを強制終了
3. タスクマネージャーも開けない場合、電源ボタンを10秒長押しで強制シャットダウン

## Q6: パスワードを忘れた

**対処法**:
1. ログイン画面で「パスワードを忘れた場合」をクリック
2. 社員番号と登録済みメールアドレスを入力
3. リセットリンクがメールに届くので、リンクから新しいパスワードを設定
4. メールにもアクセスできない場合は、社員証を持参の上、IT窓口（本館2F）へ

## Q7: Teamsの通話品質が悪い

**対処法**:
1. 有線LAN接続に切り替え（Wi-Fiは帯域が不安定）
2. カメラをオフにして帯域を節約
3. Teams → 設定 → デバイス → 音声テストで入出力を確認
4. ヘッドセットのファームウェアを最新に更新

## Q8: ファイルサーバーにアクセスできない

**対処法**:
1. VPNに接続しているか確認（社外の場合はVPN必須、手順は「VPN接続マニュアル」参照）
2. ネットワークドライブのパス `\\fileserver.example.com\shared` を確認
3. アクセス権がない場合は、上長に申請を依頼

## Q9: Windows Updateが終わらない

**対処法**:
1. 最低30分は待つ（大型アップデートは時間がかかります）
2. 30分以上進捗がない場合、PCを再起動してWindows Updateを再実行
3. エラーが表示される場合、エラーコードを控えてITヘルプデスクに連絡

## Q10: USBメモリが認識されない

**原因**: セキュリティポリシーにより、個人所有のUSBメモリは使用禁止です。

**対処法**:
- ファイルの受け渡しには社内ファイルサーバーまたはTeamsのファイル共有を使用してください
- やむを得ずUSBメモリが必要な場合は、IT部門から暗号化付きUSBメモリを貸し出します（申請制）
