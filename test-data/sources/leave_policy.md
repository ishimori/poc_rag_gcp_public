# 有給休暇規定

制定日: 2023-04-01
最終改訂: 2025-10-01
管理部門: 人事部
閲覧権限: 全社員

---

## 第1条 目的

本規定は、労働基準法に基づく年次有給休暇の付与・取得・管理について定めるものである。

## 第2条 付与日数

| 勤続年数 | 付与日数 |
|---------|---------|
| 6ヶ月 | 10日 |
| 1年6ヶ月 | 11日 |
| 2年6ヶ月 | 12日 |
| 3年6ヶ月 | 14日 |
| 4年6ヶ月 | 16日 |
| 5年6ヶ月 | 18日 |
| 6年6ヶ月以上 | 20日 |

## 第3条 申請方法

1. 社内ポータルの「勤怠管理」画面から申請
2. 取得希望日の3営業日前までに申請すること
3. 上長の承認をもって取得確定とする
4. 半日単位（午前休・午後休）での取得も可能

## 第4条 時間単位年休

1. 年間5日分（40時間）を上限に、1時間単位で取得可能
2. 申請は当日の始業前までに行うこと
3. 社内ポータルの「時間休申請」から手続き

## 第5条 計画年休

1. 会社は労使協定に基づき、年間5日を計画年休として指定できる
2. 計画年休日は年度初めに通知する
3. 2026年度の計画年休: 5/1, 8/13, 8/14, 12/29, 12/30

## 第6条 有給休暇の繰越

1. 未使用の有給休暇は翌年度に繰り越せる
2. 繰越の上限は20日とする
3. 繰越分から優先的に消化される

## 第7条 取得奨励

1. 年間10日以上の取得を推奨する
2. 取得率が50%未満の社員には、上長から取得促進の面談を実施
3. 年度末に残日数が10日以上の場合、人事部から取得計画の提出を求める場合がある
