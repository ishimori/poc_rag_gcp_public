# 給与規定

制定日: 2023-04-01
最終改訂: 2026-01-01
管理部門: 人事部
閲覧権限: 管理職以上（部長級以上）

---

**注意: 本文書は機密情報を含みます。閲覧権限のない者への共有を禁止します。**

## 第1条 目的

本規定は、当社の給与体系・昇給基準・賞与算定方法について定めるものである。

## 第2条 給与体系

| 等級 | 職位 | 月額基本給（範囲） | 役職手当 |
|------|------|-------------------|---------|
| G1 | 一般社員（新卒〜3年目） | ¥220,000 〜 ¥280,000 | — |
| G2 | 一般社員（4年目〜） | ¥280,000 〜 ¥350,000 | — |
| G3 | 主任・リーダー | ¥350,000 〜 ¥420,000 | ¥20,000 |
| G4 | 課長 | ¥420,000 〜 ¥520,000 | ¥50,000 |
| G5 | 部長 | ¥520,000 〜 ¥650,000 | ¥80,000 |
| G6 | 本部長・執行役員 | ¥650,000 〜 ¥900,000 | ¥150,000 |

## 第3条 昇給基準

1. 昇給は年1回（4月）
2. 人事評価（S/A/B/C/D）に基づき昇給額を決定
3. 標準昇給率: 評価B（標準）で基本給の2.5%

| 評価 | 昇給率 |
|------|--------|
| S | 5.0% |
| A | 3.5% |
| B | 2.5% |
| C | 1.0% |
| D | 0%（昇給なし） |

## 第4条 賞与

1. 支給時期: 6月（夏季）、12月（冬季）
2. 算定期間: 夏季＝前年10月〜3月、冬季＝4月〜9月
3. 標準支給月数: 年間4.0ヶ月分（夏季2.0、冬季2.0）
4. 業績連動係数: 会社業績に応じて0.8〜1.2倍の範囲で変動

## 第5条 諸手当

| 手当名 | 金額 | 支給条件 |
|--------|------|---------|
| 通勤手当 | 実費（上限¥50,000/月） | 全社員 |
| 住宅手当 | ¥20,000〜¥40,000 | 世帯主のみ |
| 家族手当 | 配偶者¥15,000、子1人¥10,000 | 扶養家族あり |
| リモートワーク手当 | ¥5,000/月 | 月10日以上在宅の場合 |
| 資格手当 | ¥5,000〜¥30,000 | 会社指定資格保有者 |
